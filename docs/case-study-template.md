# Case Study Template

This document defines the structural template for each of the three project tiers.

## Tier 1: Flagship (18 Sections)
1. **Executive Summary**: 100-150 words overview.
2. **Key Takeaways**: Exactly 3 bullets.
3. **What This Project Taught Me**: Highlighted reflection block.
4. **Context**
5. **The Problem**
6. **Why It Mattered**
7. **Research**
8. **Architecture**: Scan-friendly lists + SVG flowcharts.
9. **Key Decisions**: Dated chronology of changes.
10. **Development Timeline**
11. **Decision Log**
12. **Trade-offs**: 3-column Table (Decision | Benefit | Cost).
13. **What Didn't Work**
14. **Security**
15. **Performance**
16. **Monitoring**
17. **Scalability**
18. **Outcome**
19. **Lessons**
20. **Things I'd Do Differently**
21. **If I Rebuilt It Today**
22. **Questions I'm Still Exploring**
23. **Further Reading** & **Related Source Code**

## Tier 2: Medium (6 Sections)
1. **Executive Summary**
2. **Goals**
3. **Architecture**
4. **Challenges**
5. **Outcome**
6. **Lessons**

## Tier 3: Concise (5 Sections)
1. **Client Requirements**
2. **Why WordPress**
3. **Customizations**
4. **Deployment**
5. **Lessons**
