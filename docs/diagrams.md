# Diagrams & Visuals Guide

This guide documents the design standards for all diagrams and visuals.

## Standards
- **SVG Format**: Prefer responsive inline SVG structures.
- **Searchable Text**: Keep diagram text indexable using standard SVG text nodes rather than raster images.
- **Minimal Colors**: Maintain a dark-mode theme matching the website palette (dark background, light borders, subtle blue accents).
- **Architecture over Screenshots**: Use diagrams to illustrate flow and relationships. Do not add generic screenshots of code or user interfaces unless they add actual operational clarity.
